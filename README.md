## Description
NetPractice is a networking project from the 42 curriculum. The goal is to solve various networking puzzles by configuring IP addresses, subnet masks, and routing tables to make different nodes communicate successfully.

## Instructions
1. Open your terminal and navigate to the net_practice.
2. Launch the training interface by executing the startup script:
   ```bash
   ./run.sh
	```
3. On the interface, fill your login and click on **Start!**.
4. For each level, look at the network diagram.
5. Fill in the missing values for the IP addresses, masks, or routing tables.
6. Click the check button to validate the configuration and pass the level.
7. To export the configuration, click on **Get my config** button.

## Resources

The networking courses I took during my bachelor's degree and my peers at 42 helped  me a lot for this project.

This project covered the following basic networking concepts:
* **TCP/IP Addressing:** The system of unique numbers used to identify and locate devices on a network.
* **Subnet Masks:** How to divide a network and find the range of available IPs.
* **Default Gateways:** The exit point used by a local network to access the outside world.
* **Routers and Switches:** Switches connect devices in the same network, while routers connect different networks together.
* **OSI Layers:** Understanding how data travels through different levels to find and fix connection problems.
