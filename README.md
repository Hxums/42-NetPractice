## Description
NetPractice is a networking project from the 42 curriculum. The goal is to solve various networking puzzles by configuring IP addresses, subnet masks, and routing tables to make different nodes communicate successfully.

## Instructions
1. Open the NetPractice training interface in your web browser.
2. For each level, look at the network diagram.
3. Fill in the missing values for the IP addresses, masks, or routing tables.
4. Click the check button to validate the configuration and pass the level.
